# Historias que Venden

Landing page para presentar el proyecto "Historias que Venden", una propuesta visual y narrativa enfocada en comunicar testimonios, servicios y contenido de valor de forma clara y atractiva.

## Vista general

Este proyecto es un sitio web estático construido con HTML, CSS embebido y recursos visuales locales. Está pensado para publicarse fácilmente en plataformas como Netlify.

## Estructura

```txt
.
├── index.html
├── historias_que_venden.html
├── assets/
│   └── images/
├── qr-historias-que-venden.png
└── README.md
```

## Cómo verlo localmente

Abre `index.html` directamente en el navegador.

También puedes usar una extensión como Live Server en VS Code para levantar una vista local.

## Deploy

El proyecto fue desplegado con Netlify:

[Ver sitio en vivo](https://rad-snickerdoodle-73602f.netlify.app/)

![QR del sitio Historias que Venden](qr-historias-que-venden.png)

Las carpetas y archivos generados para publicación, como `deploy/`, `_zip_check/`, `.netlify/` y archivos `.zip`, no forman parte del código fuente y están excluidos del repositorio.
